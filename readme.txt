== Description ==

BuddyPress  Avatar Hover let's you add a pop box when hovering on the group/member avatars and gives you more information at a glance.
if you install bp-cover plugin , bp-pop show cover of memeber/group

== Installation ==
= To Install: =

1.  Download the plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/bp-pop/` folder to the `/wp-content/plugins/` folder on your site
4.  Single-site BuddyPres go to Plugins menu and activate there.
5.  For Multisite visit Network Admin -> Plugins and Network Activate it there.
